# BME450-project
  "Predicting coffee sales on weather and location"

# Team Members
    Sammie Stebbings (sstebbi)

# Project Description
  For this project I plan to use the dataset "Global Coffee Sales vs Weather & Dynamic Pricing", created by Moez Ali Khan, to create a neural network capable of predicting the total coffee sales for a specific location based on the weather for that day. This will require the user to input a specific date, location, and weather conditions and the model will predict what the total coffee sales will be. The dataset contains timestamps for each purchase that will be adjusted to be entered in a "month/day/year" format. Additionally, this will require some pre-processing of the dataset to convert individual purchases into groups of purchases by date and location. If this neural network can be completed in a timely manner, additional filtering options may be added to the adhance the predictions, such as inputting the desired store type or including sales of other items like tea or pasteries.
